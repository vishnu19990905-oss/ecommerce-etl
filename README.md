# Ecommerce ETL Pipeline

This project implements an end-to-end **batch ETL pipeline** for an ecommerce
dataset using **PySpark**.

The goal of this project is to demonstrate **core data engineering concepts**
such as data ingestion, transformation, and storage in an analytics-ready format.

## Tech Stack
- Python
- PySpark
- SQL
- GitHub

🚧 Work in progress — pipeline and transformations are under development
